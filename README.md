# MailAuthService представляет собой проект, который состоит из нескольких модулей

1) Модуль регистрации  пользователей
2) Нативный модуль для отправки сообщений
3) Модуль для отправки сообщений с использованием готовых инструментов
Общение между ними просиходит по кафке, а для кеширования используется редис

Каждый модуль завёрнут в докер, а также в кубер, то есть для каждого сервиса был написан свой манифест и развёрнут на подах

Для применения всех манифестов

```bash
 kubectl apply -f k8s/
 ```
Посмотреть все поды
```bash
  kubectl get pods
 ```
Чтобы отправить запрос к AuthService в Postman, необходимо прописать в терминале следующую команду

```bash
 kubectl port-forward pod/authservice-847898bcf4-2j5ds 8080:8080
 ```
